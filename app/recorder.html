Welcome to Puppetry Recorder

Puppetry Recorder we use to record user interactions on the target web pages as a test suite. To start you need to type in the toolbar the URL of a page and press Enter. As pages loaded you can interact with them (click elements, type in inputs, resize window, scroll, navigate pages and so on).

Available tools

  • Address bar - navigate to the provided URL
  • Select device combobox - resize window to match the selected device screen
  • Create Suite button - save recording as a test suite and return to Puppetry main window
  • On the page:
    • Ctrl Shift S - add "make a screenshot" command to the log. Can be used to mark a particular step in your session
    • Right-Click - register a hovered element as a target

When moving mouse cursor on a loaded page the hovered elements are being highlighted.
